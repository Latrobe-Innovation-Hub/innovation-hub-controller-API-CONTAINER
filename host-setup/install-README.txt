Install Script Guide

Before running the install.bat script, please follow these instructions:

Right-click on the install.bat file.
Select "Run as Administrator" from the context menu. This will ensure the script has the necessary permissions to make changes to the system.
After running the script, it will proceed with the following steps:

Check and set the PowerShell execution policy to allow script execution.
Install OpenSSH and set up necessary configurations for SSH communication.
Open Port 22 for SSH communication.
Modify certain registry settings for smooth interaction with the Innovation Hub API controller.
Download and install PsTools, a set of command-line tools for system management.
Reboot the system to apply changes.
Please be aware of the following:

The script will require administrative privileges to run successfully.
During the installation process, there might be some system restarts or prompts for user confirmation.
After running the install.bat script, ensure to follow any additional instructions provided for successful setup.

If you encounter any issues or have concerns, please seek help from your IT department or a knowledgeable system administrator.
